# UmbCheckout.StarterKit.Stripe

UmbCheckout Starter Kit for Stripe is a ready-to-go pre-configured Umbraco site which can be installed in under 2 minutes using dotnet templates.

## Umbraco Version

Latest Umbraco V10 LTS
